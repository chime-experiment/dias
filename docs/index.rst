dias
================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

dias is a **d**\ata **i**\ntegrity **a**\nalysis **s**\ystem designed to be
used for automated, non-realtime data processing tasks.

A deployed dias system consists of a number of analyzers which are
automatically run by the scheduler service as configured tasks.
